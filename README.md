# Diabetes Detection Application 🩺

A Full Stack Machine Learning web application that predicts the possibility of diabetes based on patient health information.

The application uses a Machine Learning model for prediction, a Flask API for backend processing, and a React-based user interface styled with Tailwind CSS and Material UI.

---

## 🚀 Features

- Diabetes prediction using Machine Learning
- User-friendly web interface
- Real-time prediction result
- Flask REST API integration
- Responsive UI design
- Material UI components
- Tailwind CSS styling

---
